# 张磊（高瓴资本创始人）表达 DNA 调研报告（03-expression-dna）

## 关键发现摘要

1. **话语内核高度稳定**：张磊公开表达围绕一组"核心语汇"展开——长期主义、做时间的朋友、价值创造、护城河、格局观、延迟满足、重仓中国、产业赋能。这些词汇自 2005 年创业、2020 年《价值》出版到近年的公开演讲反复出现，构成其表达的第一性。
2. **古典与西方的并置**：他习惯用中国古典语汇（守正用奇、弱水三千但取一瓢、桃李不言下自成蹊、风物长宜放眼量、高筑墙广积粮缓称王）作为投资哲学的"接头暗号"，同时引用巴菲特、David Swensen、亚里士多德"第一性原理"、堂吉诃德/桑丘等西方思想，形成"东西合璧"的话语风格。
3. **自我定位为配角/赋能者**：他多次用"桑丘·潘沙"（帮助英雄成就大事）、"我们是创业者，恰巧是投资人"、"研究员""长期主义者"来形容自己，强调谦逊、服务、同行，而非中心化的"英雄叙事"。
4. **情绪基调乐观且具韧性**：面对危机、失败、竞争，他倾向于将其转化为"压力测试""试金石/照妖镜""死亡谷""穿越周期"等建设性隐喻；对成功则淡化个人功劳，强调"同行者"和"时间的奖励"。
5. **表达风格兼具文人化、哲理化与故事化**：他用排比、对偶、类比、引用、反复设问；喜欢在对话中讲具体案例（腾讯名片、京东物流、百丽数字化、海底捞组织）来承载抽象理念。
6. **高频口头禅明确**："长期主义""做时间的朋友""疯狂地创造长期价值""选择与谁同行，比要去的远方更重要""做难而正确的事""流水不争先，争的是滔滔不绝"等出现频率极高。

## 详细调研

### 一、高频词汇与核心意象

张磊的公开表达存在一组高度重复的"关键词云"，几乎贯穿其著作、演讲和访谈：

| 高频词汇/短语 | 典型出处 | 可信度 |
|---|---|---|
| 长期主义 / 做时间的朋友 | 《价值》全书；2020 张磊老友记；FSDC 访谈 | 高 |
| 价值创造 / 为社会创造长期价值 | 《价值》；BEYOND Expo 2022 演讲 | 高 |
| 护城河（动态护城河） | 《价值》；2020 张磊老友记 | 高 |
| 格局观 | 《价值》；Baidu Baike 语录；Goldman Sachs 访谈笔记 | 高/中 |
| 重仓中国 | 《价值》；公开报道 | 高 |
| 产业赋能 / DVC（Deep Value Creation） | BEYOND Expo 2022 演讲；高瓴公开资料 | 高 |
| 延迟满足 | 《价值》读书笔记/金句汇编 | 中 |
| 人、生意、环境、组织 | 《价值》；各类读书笔记 | 高 |
| 与谁同行 / 靠谱的人 / 有意思的事 | 2020 张磊老友记；FSDC 访谈 | 高 |
| 做难而正确的事 | 2020 张磊老友记 | 高 |

**核心意象**：
- **"时间"**：时间的朋友、穿越周期、延迟满足、流水不争先。
- **"护城河"**：从静态品牌/专利升级为"企业不断疯狂地创造长期价值的能力"。
- **"火"**："三把火"——知识、能力、价值观（"凡是能被火烧掉的都不重要"）。
- **"镜子/试金石"**：危机是"试金石，也是照妖镜"。
- **"诗与远方"**：在苟且当下时保持"诗和远方"。

### 二、句式与修辞特征

#### 1. 对偶/排比
- "守正用奇；弱水三千，但取一瓢；桃李不言，下自成蹊。"（《价值》）
- "风物长宜放眼量；高筑墙，广积粮，缓称王。"（《价值》及多次演讲）
- "一方面 Think big think long——谋大局，思长远；另一方面用显微镜研究生意的本质。"（《价值》读书笔记）

#### 2. 设问与回答
在《价值》和老友记中，他经常先提出问题，再给出自己的答案：
- "价值是什么？首先价值不是什么……不是零和游戏，不是风口理论。"（2020 张磊老友记）

#### 3. 类比与隐喻
- 把企业护城河比作"不断地疯狂地创造长期价值的能力"。
- 把危机比作"试金石""照妖镜""压力测试"。
- 把早期创业比作"穿越死亡谷"。
- 把投资人与企业家关系比作"桑丘·潘沙与堂吉诃德"。

#### 4. 中西并置
在同一语境下，他会同时调用东西方话语：
- "第一性原理"（亚里士多德）+ "守正用奇"（老子《道德经》）。
- "做时间的朋友"（西方长期主义）+ "流水不争先，争的是滔滔不绝"（中国古语）。

### 三、古典诗词/名言引用习惯

张磊引用古典诗文有四个特点：

1. **高频三句话**：
   - "守正用奇"——化用《道德经》"以正治国，以奇用兵"。
   - "弱水三千，但取一瓢"——出自《红楼梦》/佛经典故，被其用来表达"自我约束、专注机会"。
   - "桃李不言，下自成蹊"——出自《史记·李将军列传》，表达"价值自然显现，不必喧嚣"。

2. **毛泽东诗词**：
   - "风物长宜放眼量"（《七律·和柳亚子先生》）——用于强调长期视角。
   - "高筑墙，广积粮，缓称王"——朱元璋战略，被其用来建议创业者谋定后动。

3. **投资界经典**：
   - 本杰明·格雷厄姆"市场短期是投票机，长期是称重机"。
   - David Swensen 的受托人责任与理性诚实。

4. **文学隐喻**：
   - 堂吉诃德与桑丘·潘沙：张磊自比桑丘，"我的角色就是帮助英雄人物成就大事"。（来源：大公报 2021-04-12 报道）

### 四、写作/讲话风格

#### 1. 文人化
张磊的表达带有浓厚的"人文气息"：喜欢引用诗词、历史典故、哲学概念，强调"心灵宁静""诗和远方""人文关怀"。

#### 2. 哲理性
他不满足于讲投资技巧，而是把投资上升到人生观、价值观：
- "长期主义不仅仅是一种方法论，更是一种价值观。"
- "真正的投资有且只有一条标准，那就是是否在创造真正的价值，这个价值是否有益于社会的整体繁荣。"

#### 3. 故事化
他习惯用具体案例承载抽象理念：
- 腾讯：义乌小商贩名片上印着 QQ 号。
- 京东："要么拿走三个亿，要么一分别拿"。
- 百丽：私有化后数字化转型、RFID、精益制造。
- 海底捞："双手改变命运"、组织冰山模型。

#### 4. 谦逊/去中心化
他反复淡化自己作为投资人的主角地位，强调：
- "我们是创业者，恰巧是投资人。"
- "找一帮你喜欢的真正靠谱的人，一起做有意思的事。"
- "选择与谁同行，比要去的远方更重要。"

### 五、情绪模式

#### 1. 谈成功：归因于时间、同行者与时代
- "只要不断地疯狂地创造长期价值，社会早晚会给予回报。"（2020 张磊老友记）
- "高瓴这帮人是一群乌合之众，但大家有一个共同的信念。"

#### 2. 谈失败：转化为学习契机
- 在《价值》中坦承早年因过早卖出某互联网巨头股票而错失数十倍收益（虽未点名具体公司，但属于其公开自我反思）。（来源：多家书评转述，可信度中）
- 危机时："危机真的是试金石，也是照妖镜，能照出来你的真心在哪里。"

#### 3. 谈竞争：正和游戏而非零和
- "价值的反面是零和游戏和风口论。"
- "有竞争才有进步，否则全是垄断的，没人给你好的服务。"

#### 4. 谈时代机遇：乐观主义
- 他在 FSDC 访谈中自称为乐观主义者（虽未直接说"我是乐观主义者"，但通篇基调积极）。
- "在这个时代，创新层出不穷，能生活在这个时代很幸运。"（Goldman Sachs 访谈笔记，可信度中）

### 七、反讽、幽默与口头禅

- **"乌合之众"**：本为贬义，他用来形容早期团队，带有自嘲和亲和力。
- **"人傻钱多"**：转述外界对高瓴投资京东的调侃，"高瓴这样人傻钱多的都来了"。
- **"托们"**：在《价值》发布对话中笑称在场朋友是"托们"。
- **"命题作文"**：张勇调侃"主要还是看《价值》嘛"，张磊接"对对对，命题作文"。
- **口头禅**：
  - "疯狂创造价值"
  - "做时间的朋友"
  - "选择与谁同行，比要去的远方更重要"
  - "做难而正确的事"
  - "流水不争先，争的是滔滔不绝"
  - "长期主义是一种生活方式"

## 信息缺口

1. **早期一手音视频稀缺**：2005—2015 年间张磊的公开演讲、央视《对话》、博鳌/亚布力论坛完整 transcript 较难获取，现有材料多为媒体摘要或二手转述。
2. **英文访谈原稿有限**：除 FSDC、BEYOND Expo、Goldman Sachs 访谈笔记外，Bloomberg/CNBC 的完整视频或 transcript 尚未找到，限制了对他在国际媒体中表达风格的评估。
3. **《价值》具体章节页码**：本次调研主要依赖网络摘录和读书笔记，若能获取正式出版物电子版或纸质书，可更精确地标注页码与上下文。
4. **失败案例细节**：张磊在《价值》中提到过早卖出某互联网巨头股票等失败经历，但未公开具体公司名称、时间与金额，难以做更深度的"失败表达"分析。
5. **非公开场合表达**：高瓴内部会议、致投资者信（LP letter）等未公开材料无法获取，可能包含与公开表达不同的语气、措辞或更直白的风险讨论。
6. **他人评价中的"张磊话语"**：被投企业家（马化腾、刘强东、董明珠等）对张磊的直接评价多散见于新闻报道，系统性不足。

# 05-decisions：张磊重大投资决策深度调研

> 调研维度：关键投资决策 / 决策背景、逻辑、结果与反思
> 调研时间：2026-06-26
> 适用对象：张磊（高瓴资本创始人）思维操作系统 Skill

---

## 一、关键发现摘要

1. **“All in” 式开局**：2005 年张磊创立高瓴后，将耶鲁捐赠基金支持的几乎全部资金（约 2000 万–3000 万美元）重仓腾讯，奠定高瓴“长期结构性价值投资”的底色。这笔投资后来成为其最常被引用的“一战封神”案例。
2. **从“发现价值”到“创造价值”的跃迁**：2010 年投资京东是高瓴从二级市场走向一级市场、从财务投资走向产业赋能的关键转折。张磊以 3 亿美元“硬塞”给刘强东，逻辑是京东有机会成为“亚马逊 + UPS”的结合体。
3. **传统产业数字化改造范式**：2017 年主导百丽国际私有化（531 亿港元），是高瓴“哑铃理论”与深度赋能（DVC）的集中实践；2019 年 417 亿元入股格力电器，则是其参与国企混改、重仓中国制造的标志性交易。
4. **“重仓中国”与“做时间的朋友”两条主线**：张磊的关键决策多围绕中国消费升级、互联网基础设施、先进制造与产业数字化展开；其公开反思强调“长期主义”并非死拿，而是对企业创造真实价值的陪伴。
5. **信息缺口明显**：高瓴早期很多交易的实际金额、入股价格、退出时点与真实回报并未完全公开；美团、宁德时代、Zoom、Airbnb、拼多多、字节跳动等项目的具体投资轮次与回报多依赖媒体报道或招股书间接推算，可信度参差不齐。

## 二、详细调研

### 2.1 2005 年：创立高瓴资本

| 维度 | 内容 |
|------|------|
| **背景** | 张磊 1990 年考入中国人民大学国际金融专业，1998 年赴耶鲁攻读 MBA，曾在耶鲁捐赠基金实习并师从 David Swensen，后任职纽约证券交易所首任中国首席代表。2005 年中国股权分置改革启动、《合伙企业法》即将修订，私募股权投资迎来制度窗口期。 |
| **选择** | 2005 年 6 月回国创立 Hillhouse Capital（中文名“高瓴”，取自“高屋建瓴”）。初期获得耶鲁捐赠基金 2000 万美元投资（部分资料称 3000 万美元，可能包含后续追加或张磊个人/团队出资），耶鲁成为其第一个 LP。 |
| **逻辑** | 张磊认为中国资产价值被严重低估，互联网与新经济刚刚起步，长期结构性机会巨大；同时希望将耶鲁捐赠基金的长期资本模式引入中国。高瓴官网自述其自创立起即“重仓中国”。 |
| **结果** | 高瓴从 2000 万美元起步，管理规模在随后 15–20 年间扩张至数千亿人民币（公开报道称 5000 亿–6000 亿人民币或 600 亿美元级别），成为亚洲规模最大的私募基金之一。 |
| **事后反思/表述** | 张磊在《价值》自序中称：“人生的每一次选择都是一次重要的价值判断，而每一次判断都来源于人们的底层信念。”创立高瓴被视为其对“长期主义”价值观的第一次重大实践。 |
| **来源与可信度** | 高瓴官网 / 百度百科 / 财新《缔造投资传奇的耶鲁大学师徒》【高】；部分自媒体关于“个人出资 1000 万美元”的细节【低】 |

### 2.2 2005 年：重仓腾讯

| 维度 | 内容 |
|------|------|
| **背景** | 腾讯 2004 年 6 月在香港上市，2005 年时市值不足 20 亿美元，主要收入来自 SP/短信增值业务，QQ 用户虽庞大但盈利模式不清晰，外资机构普遍不看好。 |
| **选择** | 高瓴将可支配的几乎全部资金（约 2000 万–3000 万美元）重仓腾讯股票，成为其早期重要机构股东之一。 |
| **逻辑** | 张磊通过一手调研发现，义乌等中小商户名片上普遍印有 QQ 号，说明 QQ 已深度嵌入中国商业社会的沟通基础设施；他认为腾讯拥有“社交网络的低成本连接价值”，未来变现空间巨大。 |
| **结果** | 腾讯市值此后 10 余年增长数百倍，高瓴这笔投资公开报道称回报 200 倍–400 倍不等，是高瓴声誉与资金积累的第一桶金。 |
| **事后反思/表述** | 张磊在《价值》中反思：早年曾因过早卖出某互联网巨头股票而错失数十倍收益，这让他更坚定长期持有。《价值》中还提到，真正的价值投资要看到企业创造的社会价值，而非短期盈利。 |
| **来源与可信度** | 百度百科 / 36氪 / 虎嗅 / 雪球等多篇报道交叉验证投资逻辑与回报量级【高】；具体回报倍数因未披露退出细节而难以精确核实【中】 |

### 2.3 2010 年：投资京东

| 维度 | 内容 |
|------|------|
| **背景** | 2010 年京东连续亏损多年，重资产自建物流模式不被投资界看好；刘强东向 30 余家机构寻求融资均遭拒绝，初步向张磊提出融资 7500 万美元。 |
| **选择** | 张磊提出“要么投 3 亿美元，要么一分不投”，最终高瓴以约 2.65 亿–3 亿美元投资京东，成为当年中国互联网最大单笔早期投资。 |
| **逻辑** | ① 张磊算过账：在一线城市搭建仓储物流体系至少需要 25 亿元人民币，7500 万美元“连实验都不够”；② 京东有机会成为“亚马逊 + UPS”的结合体——贝索斯最大的遗憾是亚马逊成立时 UPS 已很强大，而中国没有这样的对手；③ 电商竞争最终是供应链效率与用户体验的竞争，资本可以加速核心竞争力的淬炼。 |
| **结果** | 2014 年京东纳斯达克上市时，高瓴 3 亿美元投资账面价值升至约 39 亿美元；京东物流 2021 年港股独立上市，成为京东核心护城河。 |
| **事后反思/表述** | 张磊在《价值》中回忆：投京东不仅是“赌物流赛道”，更是“赌刘强东这个人”——他的长期主义与执行力是关键。张磊还曾复盘：“3 亿美元在当时是很大，但高瓴在当时的管理规模已是几十亿美金，即便这笔投资全军覆没，高瓴也可以 take the risk。” |
| **来源与可信度** | 《价值》原文节选 / 36氪 / 新浪财经 / 投资界 / 虎嗅，刘强东融资需求 7500 万美元、3 亿美元投资额、京东上市后 39 亿美元估值等细节多源交叉【高】；具体回报 300 亿美元等说法为自媒体夸张【低】 |

### 2.4 2008/2010 年：投资蓝月亮

| 维度 | 内容 |
|------|------|
| **背景** | 2008 年全球金融危机期间，张磊研究中国消费升级，发现家用洗涤市场被宝洁、联合利华的洗衣粉占据，洗衣液市占率不足 4%–5%；而美、日洗衣液已是主流。 |
| **选择** | 张磊主动联系蓝月亮创始人罗秋平，建议其从洗手液切入洗衣液赛道；2010 年高瓴以 4500 万美元独家投资蓝月亮 A 轮，2011 年追加约 103 万美元，合计持股约 10%。 |
| **逻辑** | ① 跨国公司有历史包袱，认为中国人均 GDP 不足 3000–4000 美元，洗衣液市场不成熟；② 张磊判断中国消费升级与洗衣机保有量提升将推动“粉转液”；③ 蓝月亮可借助先发优势与营销投入建立“洗衣液 = 蓝月亮”的心智。 |
| **结果** | 蓝月亮 2010 年末洗衣液市占率达 44%，2020 年 12 月香港上市首日市值约 855 亿–900 亿港元，高瓴持股市值约 85 亿港元，账面回报约 24 倍。但 2021 年后蓝月亮利润连续下滑，2024 年出现亏损，市值大幅缩水。 |
| **事后反思/表述** | 张磊在《价值》中将蓝月亮作为“新兴市场本土品牌战胜跨国公司品牌”的经典案例，强调“基于长期研究所带来的对趋势变化的把握”。 |
| **来源与可信度** | 新浪财经 / 网易财经 / 搜狐财经交叉验证 4500 万美元投资额与市占率【高】；上市首日市值与回报倍数来自公开招股书与媒体报道【中】 |

### 2.5 2017 年：百丽国际私有化

| 维度 | 内容 |
|------|------|
| **背景** | 百丽国际曾是中国最大女鞋零售商，2014 年后受电商冲击业绩下滑，2017 年股价较峰值跌去近 80%，决定私有化退市。 |
| **选择** | 2017 年 4 月，由高瓴资本、鼎晖投资、智者创业（管理层持股平台）组成的财团以 531 亿港元私有化百丽国际；高瓴持股约 56.81%，成为控股股东，张磊出任董事长。 |
| **逻辑** | ① 张磊认为“没有哪个失败的企业每年能有几十亿元的现金流”，百丽拥有庞大的线下零售网络、供应链与品牌基础；② 运动鞋服业务（滔搏）是被低估的优质资产；③ 可通过数字化改造提升库存周转与渠道效率，实现“3 倍回报”。 |
| **结果** | 2019 年 10 月滔搏从百丽分拆上市，首日市值约 574 亿–700 亿港元，已超过百丽私有化总价；百丽时尚 2022 年递交港股 IPO 申请。张磊公开表示该交易已实现“3 倍回报”的三分之二。 |
| **事后反思/表述** | 张磊在《价值》中称，收购百丽后坚持“锦上添花、务实再务实、小步快跑”，在百丽能力与基因上做加法，充分信任原有管理层。他还引用百丽创始人邓耀的话表达对企业家精神的尊重。 |
| **来源与可信度** | 36氪 / 创业邦 / 投资界 / 雪球，531 亿港元私有化金额、高瓴持股比例、滔搏上市市值等均有公开披露【高】；“3 倍回报”为张磊公开表述【中】 |

### 2.6 2019 年：投资格力电器（国企混改）

| 维度 | 内容 |
|------|------|
| **背景** | 2019 年 4 月，珠海市国资委旗下格力集团宣布转让格力电器 15% 股权，引入战略投资者，25 家机构参与角逐。 |
| **选择** | 2019 年 10 月，高瓴资本通过珠海明骏投资合伙企业（有限合伙）以 416.62 亿元（每股 46.17 元）受让格力电器 15% 股权，成为第一大股东；格力电器变为“无实际控制人”状态。 |
| **逻辑** | ① 格力电器是中国制造业名片，空调龙头，现金流与分红稳健；② 高瓴希望借鉴百丽数字化经验，推动格力智能制造、国际化与多元化；③ 参与国企混改，支持实体经济转型升级。 |
| **结果** | 高瓴入股后股价长期低于成本，2022 年前后曾浮亏约 25%–30%；依靠高分红摊薄成本，截至 2025 年部分测算显示微利或基本持平，年化回报远低于高瓴历史平均水平。高瓴至今未向格力董事会派驻董事。 |
| **事后反思/表述** | 张磊曾对董明珠承诺：不干预日常经营、3 年内不减持、推动不超过 4% 的股权激励。外界评价：高瓴“资本想快、实业想慢”的逻辑与董明珠的强管理风格存在张力，格力的“无实际控制人”结构实际形成“高瓴出钱 + 董明珠实控”的微妙平衡。 |
| **来源与可信度** | 格力电器公告 / 财新 / 36氪 / 新浪财经 / 雪球，交易金额、价格、条款等来自公开披露【高】；浮亏与收益测算为媒体/投资者估算【中】 |

### 2.8 投资失败与公开反思

张磊在《价值》中坦诚分享过失败教训，主要包括：

1. **过早卖出优质资产**：早年因过早卖出某互联网巨头股票而错失数十倍收益，使其深刻认识到“长期主义的对立面不仅是投机，更是对自身认知局限的妥协”。
2. **低估传统行业转型潜力**：曾因低估传统行业的转型空间而与潜在机会失之交臂，这也成为后来重仓百丽、格力等传统产业升级的心理背景。
3. **格力案例的争议**：格力投资被视为张磊“长期主义”理念的一次压力测试。截至 2025 年，该交易账面回报平平，外界质疑高瓴在格力治理中缺乏实质影响力，也反映出“赋能型资本”在强势企业家面前的边界。

> **来源与可信度**：《价值》原文关于失败案例的反思【高】；格力投资的评价与争议来自媒体分析【中】。

## 四、信息缺口

1. **早期交易细节缺失**：高瓴 2005 年腾讯投资的具体股数、成本价、后续减持节奏从未完整披露；京东 2010 年投资的确切金额（2.65 亿或 3 亿美元）存在不同报道。
2. **真实回报难以核实**：腾讯、京东、蓝月亮、百丽等案例的回报倍数在媒体中差异较大（200 倍、400 倍、13 倍、24 倍等），多含估算或自媒体夸张，缺乏经审计数据。
3. **美团、宁德时代、Zoom、Airbnb、拼多多、字节跳动的投资档案不完整**：具体轮次、投资金额、进入估值、退出比例等大多未公开，主要依赖招股书间接推算与 13F 持仓变动。
4. **张磊个人决策过程的一手资料有限**：除《价值》与少数演讲外，张磊对具体项目的内部讨论、投委会分歧、与企业家谈判细节鲜少公开。
5. **失败案例披露不足**：张磊在《价值》中提到过早卖出与低估传统行业的教训，但未点名具体公司；高瓴部分浮亏项目（如格力、隆基、部分中概股）的反思多来自外界评论，而非其本人系统复盘。
6. **近期动态（2023–2026）跟踪不足**：高瓴在 AI、半导体、新能源等领域的新布局，以及张磊本人国籍、办公地点等个人话题，虽舆论关注度高，但与其投资决策核心逻辑关联有限，本报告未重点展开。

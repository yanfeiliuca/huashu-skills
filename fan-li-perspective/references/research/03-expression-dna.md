# 03-expression-dna｜言论风格与商业格言

## 核心发现（3–5 句）

范蠡在可靠史料中的言论呈现高度结构化、对仗化与农事/天文意象密集的特征。《国语·越语下》的谏辞多为四言、五言韵文，善用“天—地—人—时”三元框架与“水/舟/农事/狩猎”比喻；《史记·货殖列传》所载经济格言则把价格周期、物资储备与资金流转具象化为“旱则资舟，水则资车”“贵出如粪土，贱取如珠玉”“财币欲其行如流水”。[Source: https://ctext.org/guo-yu/yue-yu-xia/zh][Source: https://ctext.org/text.pl?node=9234&if=gb&show=parallel] 这些表达既非范蠡亲笔，也不是现代意义上的“商战语录”，而是司马迁对“计然—范蠡”传统的整理。后世托名的《陶朱公生意经》《范子计然》以及“商圣”“三聚三散”“文财神”等标签，成书或形成年代较晚，不能当作范蠡本人的表达 DNA。[Source: https://cchc.fah.um.edu.mo/wp-content/uploads/2018/08/262.pdf][Source: https://ctext.org/dictionary.pl?if=gb&did=2300] 因此，构建“范蠡视角”时应以先秦两汉史料中“韵文化、周期性、谦逊而决绝”的话语结构为基础，后世格言仅可作为民间商业文化参照。

## 一、可靠史料中的范蠡原话与早期格言

### 1.1 《国语·越语下》谏言原话

《国语·越语下》是记录范蠡与勾践对话最集中的先秦文本，但学界指出其文风与《越语上》《吴语》明显不同，可能是后人运用公共素材改写、以凸显范蠡谋臣形象的产物。[Source: https://www.cl.ntu.edu.tw/uploads/root/%E4%B8%AD%E5%9C%8B%E6%96%87%E5%AD%B8%E7%A0%94%E7%A9%B6/50-2.pdf] 其中反复出现的核心框架是“持盈—定倾—节事”三元：

> “夫国家之事，有持盈，有持倾，有节事。”王曰：“为三者，奈何？”对曰：“持盈者与天，定倾者与人，节事者与地。”[Source: https://ctext.org/guo-yu/yue-yu-xia/zh]

同一篇中，范蠡用天道、时势、阴阳、四时等自然节律论证政治军事决策：

> “天道盈而不溢，盛而不骄，劳而不矜其功。夫圣人随时以行，是谓守时。天时不作，弗为人客；人事不起，弗为之始。”[Source: https://ctext.org/guo-yu/yue-yu-xia/zh]

在勾践急于伐吴时，范蠡强调等待时机的重要性：

> “时不至，不可强生；事不究，不可强成。”[Source: https://ctext.org/guo-yu/yue-yu-xia/zh]
> “从时者，犹救火、追亡人也，蹶而趋之，惟恐弗及。”[Source: https://ctext.org/guo-yu/yue-yu-xia/zh]
> “得时无怠，时不再来，天予不取，反为之灾。”[Source: https://ctext.org/guo-yu/yue-yu-xia/zh]

灭吴后，范蠡以“君辱臣死”的臣道逻辑请辞，并以“君行制，臣行意”作结：

> “臣闻之，为人臣者，君忧臣劳，君辱臣死。昔者君王辱于会稽，臣所以不死者，为此事也。今事已济矣，蠡请从会稽之罚。”……范蠡对曰：“臣闻命矣。君行制，臣行意。”[Source: https://ctext.org/guo-yu/yue-yu-xia/zh]

### 1.2 《史记·越王勾践世家》遗大夫种书

《史记》记载范蠡去越后从齐国致信文种，以狩猎工具比喻君臣关系的可弃性：

> “范蠡遂去，自齐遗大夫种书曰：‘蜚鸟尽，良弓藏；狡兔死，走狗烹。越王为人长颈鸟喙，可与共患难，不可与共乐。子何不去？’”[Source: https://ctext.org/text.pl?node=6557&if=gb&show=parallel]

《吴越春秋》与《论衡》等书对此有变文，如“高鸟已散，良弓将藏；狡兔已尽，良犬就烹”，并增加“鹰视狼步”等体貌描写。[Source: https://ctext.org/text.pl?node=6557&if=gb&show=parallel] 各本存在差异，说明这一“格言”在汉代已有多种流传形式，并非固定文本。

### 1.3 《史记·货殖列传》中的经济格言

《史记·货殖列传》把一段关于天时、物价、供需、储备的论述归于“计然”，并说明范蠡“师事之”、治国后用其术治家：

> “昔者越王句践困于会稽之上，乃用范蠡、计然。计然曰：‘知斗则修备，时用则知物，二者形则万货之情可得而观已。故岁在金，穰；水，毁；木，饥；火，旱。旱则资舟，水则资车，物之理也。六岁穰，六岁旱，十二岁一大饥。……贵上极则反贱，贱下极则反贵。贵出如粪土，贱取如珠玉。财币欲其行如流水。’”[Source: https://ctext.org/text.pl?node=9234&if=gb&show=parallel]

同篇又记范蠡弃官经商后的自我定位：

> “范蠡既雪会稽之耻，乃喟然而叹曰：‘计然之策七，越用其五而得意。既已施于国，吾欲用之家。’……乃治产积居。与时逐而不责于人。故善治生者，能择人而任时。十九年之中三致千金，再分散与贫交疏昆弟。”[Source: https://ctext.org/text.pl?node=9235&if=gb&show=parallel]

需要注意，《汉书·货殖传》作“计然之策，十用其五而得意”，与《史记》“七策用五”不同，显示汉代文献在数字记载上已有分歧。[Source: https://ctext.org/text.pl?node=9235&if=gb&show=parallel]

### 1.4 可靠史料中的表达特点

- **高度程式化的开场白**：范蠡进谏常以“臣闻之”起头，用“我听说”降低主观武断感；又以“王不问，蠡不敢言”表明谦逊与君臣分寸。[Source: https://ctext.org/guo-yu/yue-yu-xia/zh]
- **条件式判断**：如“天时不作，弗为人客；人事不起，弗为之始”“时不至，不可强生；事不究，不可强成”，句式多为“X 则/不 X，则 Y”。
- **三元并置**：持盈/定倾/节事、天/人/地、阴/阳/四时，形成整齐的认知网格。
- **决绝而退的结尾**：“君行制，臣行意”以八字对仗完成从臣道到个人意志的转换。

## 二、后世托名著作与民间格言

### 2.1 《陶朱公生意经》《陶朱公商训》《陶朱公经商十八则》

《陶朱公生意经》又称《陶朱公商经》《陶朱公商训》或《陶朱公经商十八则》，今见文本成书年代不明，但从行文风格判断，成书时间较晚，是后人根据陶朱公经商思想加工整理而成，并非范蠡著作。[Source: https://cchc.fah.um.edu.mo/wp-content/uploads/2018/08/262.pdf][Source: https://ctext.org/dictionary.pl?if=gb&did=2300] 常见条目如：

> “生意要勤快，切勿懒惰，懒惰则百事废；价格要定明，切勿含糊，含糊则争执多；费用要节俭，切勿奢华，奢华则钱财竭；赊欠要识人，切勿滥出，滥出则血本亏……”[Source: https://www.wang1314.com/doc/topic-22122133-1.html]

这些内容体现了明清商书的训诫风格，语言通俗、条目化，与《史记》《国语》中范蠡的韵文谏辞在修辞上差异显著。

### 2.2 《范子计然》

《范子计然》在《唐书·艺文志》中著录为“范蠡问，计然答”，列于农家，原书已佚，今见辑本多出自马总《意林》等后世类书。[Source: https://www.zgxdzy.net/ch/reader/create_pdf.aspx?falg=1&file_no=20180822&quarter_id=8&year_id=2018] 现代研究指出，其中出现的人参、犀角等药名在西汉才始用，因此该书即使成书于西汉，作者也不可能是春秋末期的范蠡；书名用“范子”“计然”二人之名，是借名人效应的托名做法，类似《黄帝内经》《神农本草经》。[Source: https://www.zgxdzy.net/ch/reader/create_pdf.aspx?falg=1&file_no=20180822&quarter_id=8&year_id=2018] Chinese Text Project 亦明确标注“《范子计然》出自唐马总的《意林》一书，作者也非范蠡”。[Source: https://ctext.org/dictionary.pl?if=gb&did=2300]

### 2.3 “商圣”“三聚三散”“文财神”等后世标签

- **“商圣”**：并非先秦两汉文献中的称号，而是后人追赠。清代以前多数著述以计然为范蠡之师；现代地方志与通俗读物称范蠡“被后人尊为商圣”“被史学界称为……商家鼻祖”。[Source: https://www.lushixian.gov.cn/3463/616994208/1897414.html]
- **“三聚三散”**：语源来自《史记·货殖列传》“十九年之中三致千金，再分散与贫交疏昆弟”，后世概括为“三聚三散”并加入“一聚一散”“二聚二散”等叙事细节。[Source: https://ctext.org/text.pl?node=9235&if=gb&show=parallel] 这些细节（如带西施泛湖、长子少子救次子等）多出自《史记·越王勾践世家》及后世演绎，并非严格的同期记录。
- **“文财神”**：Chinese Text Project 指出“后代许多生意人皆供奉他的塑像，称之财神”。[Source: https://ctext.org/dictionary.pl?if=gb&did=2300] 地方志亦称范蠡“被中国民间恭奉为‘文财神’”。[Source: https://www.lushixian.gov.cn/3463/616994208/1897414.html] 这属于民间信仰，不能作为范蠡本人言论或思想的证据。

## 三、表达风格与修辞特征

### 3.1 韵文化与对仗韵律

《国语·越语下》中的范蠡谏辞具有明显的韵文特征。台湾大学宫瑞龙的研究指出，前人如杨伯峻、沈长云等已注意到《越语下》“多排体韵文”“语言讲求对仗韵律”，吕祖谦更称其“辞气奇峻，不类春秋时语”。[Source: https://www.cl.ntu.edu.tw/uploads/root/%E4%B8%AD%E5%9C%8B%E6%96%87%E5%AD%B8%E7%A0%94%E7%A9%B6/50-2.pdf] 例如：

> “因阴阳之恒，顺天地之常，柔而不屈，强而不刚，德虐之行，因以为常；死生因天地之刑，天因人，圣人因天；人自生之，天地形之，圣人因而成之。”[Source: https://ctext.org/guo-yu/yue-yu-xia/zh]

该段以四言句为主，韵脚密集，接近句句用韵。这种文体便于口传讲诵，但也说明《越语下》的文本可能经过后人加工，不等同于范蠡即兴口语。

### 3.2 核心意象系统

| 意象类别 | 典型表达 | 功能 |
|---|---|---|
| 天时/周期 | “岁在金，穰；水，毁；木，饥；火，旱”“六岁穰，六岁旱，十二岁一大饥” | 把经济波动归因于可预测的周期 |
| 水与舟车 | “旱则资舟，水则资车”“财币欲其行如流水” | 以水流、舟车隐喻资源储备与资金周转 |
| 农事与生长 | “时不至，不可强生；事不究，不可强成”“田野开辟，府仓实，民众殷” | 用农业节律说明人事不可违时 |
| 狩猎与工具 | “蜚鸟尽，良弓藏；狡兔死，走狗烹” | 以猎具可弃性隐喻君臣/工具关系 |
| 阴阳/日月 | “阳至而阴，阴至而阳；日困而还，月盈而匡” | 用自然对立统一说明军事进退 |
| 身体/相术 | “长颈鸟喙”“鹰视狼步” | 以体貌特征推断君主性格与风险 |

这些意象的共同点是：**把抽象的政治、经济、军事判断转译为可见的自然现象或日常劳作经验**。

### 3.3 句式与语气

- **“……者……也”定义式判断**：“持盈者与天，定倾者与人，节事者与地”“兵者，凶器也；争者，事之末也”。
- **条件式禁令**：“天时不作，弗为人客；人事不起，弗为之始”“无敢居贵”。
- **反义并置**：“贵出如粪土，贱取如珠玉”“贵上极则反贱，贱下极则反贵”。
- **先抑后扬/先退后进**：开头“蠡不敢言”“臣闻之”铺垫谦逊，结尾转为“君行制，臣行意”的决绝。
- **引用谚语**：“伐柯者其则不远”“觥饭不及壶飧”，借公共谚语增强说服力。

### 3.4 谦逊表达与确定性表达的张力

范蠡在君臣对话中常用自谦称谓“蠡”“臣”，并以“王不问，蠡不敢言”表达进退分寸；但在陈述天道、周期、市场规律时，语气极为确定，如“物之理也”“天节固然”“时不再来”。这种“对人谦、对理决”的表达方式，是其话语 DNA 的重要特征。

## 四、表达 DNA 提炼

基于可靠史料，可提炼以下“范蠡视角”的表达模板：

1. **认知框架：天地人三元 + 时势周期**
   - 任何决策先问：天时是否已至？人事是否已起？地利是否相宜？
   - 常用语式：“X 者与天/人/地”“X 则 Y，物之理也”。

2. **论证方式：以自然节律类比人事/市场**
   - 把价格波动说成“一贵一贱，极而复反”；把资金流转说成“行如流水”；把军事时机说成“阳至而阴，阴至而阳”。
   - 常用语式：“犹……也”“譬如……”“物之理也”。

3. **语气节奏：先谦后退，后发先至**
   - 开场：“臣闻之……”“王不问，蠡不敢言。”
   - 中间：条件判断、对仗排比。
   - 结尾：当断则断，如“君行制，臣行意”。

4. **格言结构：对偶、反义、条件化**
   - 贵/贱、出/取、旱/水、舟/车、盈/溢、盛/骄。
   - 例：“贵出如粪土，贱取如珠玉”“旱则资舟，水则资车”。

5. **自我定位：工具理性 + 臣道意识**
   - 把自己比作“良弓”“走狗”等可被弃置的工具，强调“功成身退”的理性。
   - 对君主：称“王”“君王”，自称“蠡”“臣”。

6. **风险表达：以狩猎/农事隐喻警示**
   - “蜚鸟尽，良弓藏”提示政治风险；“时不至，不可强生”提示时机风险；“无敢居贵”提示市场风险。

## 五、史料有限与传说成分标注

1. **《国语·越语下》的可信度**：该篇虽为先秦文献，但文风特殊，可能与《管子》《黄帝书》等存在互见，部分学者认为其经过后人改写以塑造范蠡形象。[Source: https://www.cl.ntu.edu.tw/uploads/root/%E4%B8%AD%E5%9C%8B%E6%96%87%E5%AD%B8%E7%A0%94%E7%A9%B6/50-2.pdf] 因此其中的“范蠡原话”应视为**早期文献对范蠡话语风格的理想化呈现**，而非实录。

2. **《史记》《汉书》的整理性质**：司马迁、班固去范蠡时代已数百年，所载对话与格言是汉代史家对越国传统的整理；数字记载（计然“七策”还是“十策”）已出现矛盾。[Source: https://ctext.org/text.pl?node=9235&if=gb&show=parallel]

3. **范蠡本人无亲笔著作传世**：《汉书·艺文志》兵家类著录《范蠡》二篇，今佚；《陶朱公养鱼经》等亦多属托名。[Source: https://www.zgxdzy.net/ch/reader/create_pdf.aspx?falg=1&file_no=20180822&quarter_id=8&year_id=2018]

4. **“商圣”“文财神”“三聚三散”均为后世建构**：这些标签成于民间信仰、明清商书及现代通俗文化，不能反向证明范蠡本人的表达风格或商业理念。

5. **西施随范蠡泛舟等情节**：属于后世传说，先秦两汉可靠史料未载。[Source: https://ctext.org/dictionary.pl?if=gb&did=2300]

## 决策一：从大厂离职（2023年3月）

**背景**
- 在美团等大厂做了 15 年用户研究和产品运营
- 35 岁遭遇中国科技行业的"35岁危机"
- 亲历两次公司期权承诺落空（两家不同公司）
- 评价系统持续要求他走管理岗，但他判断自己"不擅长也不喜欢管人"
- 原话："我只能做自己真正喜欢的事"

**选择**
- 2023年3月裸辞，无后路
- 不打算找新工作，开始探索 AI 领域
- 刻意维持"不安全状态"：没在北京买房，没有孩子，生活成本低

**结果**
- 初期：旅居期间用 ChatGPT 学 Python，只能做简单浏览器插件
- 视频内容早期无人问津
- 做了 20+ 个失败原型产品，无一成功
- 2024年8月 Cursor 出现，才突破瓶颈

**事后反思**
- 承认自己"浪费了三年在管理岗上，追求一个本不适合自己的职业路径"
- 大厂 15 年的用户研究经验反而成了独立开发的优势——能避免"自嗨式开发"，先分析用户真实需求

**体现的价值观**
- 强烈的自主驱动，"只做自己喜欢的事"不是随口说说，是用 15 年职场痛苦换来的结论
- 主动选择脆弱性：拒绝用房产/家庭/稳定工作来对冲风险，把"不安全"当作保持专注的条件

**背景**
- 2022-2024 年的探索期，用 ChatGPT/早期 AI 工具开发，能力有限

**体现的价值观**

- 拒绝规模化：明确表示"不想有团队，只喜欢一个人工作"，走资本路线对他"压力太大"

## 决策三：小猫补光灯定价 1 元（2024年底）

**背景**
- 女朋友需要补光灯辅助拍照，他用 AI 编程 1 小时做出了 App
- 产品上了免费分类榜 Top 20，和 YouTube、Instagram、Canva 排在一起

**选择**
- Pro 版定价 1 元（象征性收费）
- 没有选择免费+广告或订阅制

**结果**
- 付费应用榜第一，坐榜逾一个月
- 但 1 元价格几乎不赚钱，主要收益来自品牌曝光和内容素材
- 引发争议：有人不满他"不会编程却卖 AI 编程课"

**公开回应争议**
> "对的，我不会编程，我只会 AI 编程，我在卖 AI 编程的课。我看到了挺多人不爽的，不好意思，不爽的人可能还是需要忍一忍。你会编程，你最会编程了，你做出啥了？"

**事后反思（可观察到的矛盾）**
- 自称"不追求钱"，却开知识星球（365元/年，1900+成员）
- 自称"不急于变现"，却承认企业培训定价 2-10 万、演讲定价 5-8 万
- 解读：变现是被动结果，他的核心动机确实不是钱，但也没有放弃商业化

**体现的价值观**
- 定价逻辑中有强烈的"成就感驱动"而非利润驱动：上榜本身比收益重要
- 面对质疑选择正面硬刚，而不是解释和辩护——这是他一贯的风格

## 决策四：拒绝出版社，选择开源橙皮书（2025-2026）

**背景**
- 出版过一本传统书籍《一本书玩转 DeepSeek》
- 出版完成后"对出书这件事没啥光环了"
- OpenClaw 橙皮书期间，不下 5 家出版社编辑主动联系合作

**选择**
- 全部拒绝
- 改为 GitHub + PDF 免费开源发布
- 累计出版 12 本橙皮书（Claude Code、Cursor、Agent Skills、Hermes Agent、OpenClaw、Gemma、Codex 等），全部免费

**结果**
- Claude Code 橙皮书在微信读书冲上热搜第一
- 通过免费开源积累了大量 GitHub Stars 和个人品牌背书
- 间接拉动演讲、企业培训等高价值变现

**事后反思（可观察到的逻辑）**
- 免费书 → GitHub Star → 个人品牌 → 企业培训（5-8万/场）
- 这是一个有意识的"漏斗"策略，不是单纯的理想主义
- 原话："做一件有复利的事比做一百件没复利的事更重要"

**体现的价值观**
- 选择影响力优先于短期收益，认为免费传播 > 收费变现（在教育内容领域）
- 对"出书"这个虚荣指标祛魅后，选了传播效率更高的路径

## 决策五：构建 nuwa-skill（女娲，2026年初）

**背景**
- 他在做内容时已经手动提炼了多个公众人物的思维框架
- 见证了 colleague-skill（蒸馏同事）的病毒式传播
- 判断："为什么只能蒸馏同事？历史上最厉害的人都留下了大量公开信息"

**选择**
- 开发 nuwa-skill：自动化"蒸馏任何人的思维方式"的 Agent Skill
- 内置 6 条平行研究通道（书、播客、社媒、批评者、决策记录、时间线）
- 三重验证机制，防止 AI 捏造
- 首批蒸馏对象：Jobs、Musk、Munger、Naval、Feynman、Karpathy 等

**结果**
- 6 周内获得 20,000+ GitHub Stars
- 成为中文 AI 工具社区 2026 年最有影响力的开源 Skill 之一
- 衍生出大量社区用户自创的 person-skill（包括本项目所用的 huashu-perspective）

**体现的价值观**
- 工具设计哲学："基础设施思维"——不做一次性内容，做可被无限调用的结构
- 核心命题：让少数人的认知框架变成所有人都能调用的基础设施
- 明确区分：nuwa 不是"让 AI 说话像某人"，是"让 AI 用某人的方式看问题"

## 决策六：开发 darwin-skill（自动进化系统，2026年）

**背景**
- 他自己管理的 Skills 数量增长到 60+
- 手动维护 60 个 Skill 质量不可行
- 观察到现有 Skill 评审工具只检查"结构合规"，不检查"实际效果"

**选择**
- 开发 darwin-skill：9 维度自动评估 + 单维度迭代 + 人工检查点 + 回滚机制
- 灵感来源：Andrej Karpathy 的 autoresearch 方法（只保留可验证的改进，其余自动回滚）
- V2.0 引入微软研究院的 SkillLens 和 SkillOpt 框架

**结果**
- 4,300+ GitHub Stars
- 解决了 Skills 生态规模化后的质量衰减问题

**体现的价值观**
- "能自动化的事情就自动化，但人类判断力不可完全外包"——darwin 保留了人工检查点
- 复利思维的技术实现：系统能随时间自动变好，不依赖人工持续投入

## 决策七：《记忆的加与减》——将 AI 记忆手册从 93KB 砍至 22KB（2026年3月）

**背景**
- 花了几个月积累了 93KB 的 CLAUDE.md 记忆手册
- 里面塞满了偏好、项目介绍、代码规范、API 文档、历史任务记录、各种规则
- 假设："信息越全，AI 就越懂我"

**选择**
- 做了一件"反直觉的事"：将文件砍到 22KB，删掉四分之三

**核心洞察（重建自原 tweet 内容）**
- 信息密度 ≠ AI 理解质量
- 上下文越长，AI 越容易在噪音中迷失核心意图
- 精简记忆反而提升了 AI 响应的准确度和针对性

**体现的价值观**
- "减法比加法难"——这是他对 AI 工作流的反直觉判断
- 愿意推翻自己几个月的积累，以验证新假设
- 内容创作逻辑同理：他的内容风格是反向删减，而非堆砌信息

## 决策八：huashu-design 的起源——从使用 Claude Design 到"让图形工具层消失"（2026年）

**背景**
- Anthropic 发布 Claude Design 后，他花大量时间使用
- 使用后发现自己"没有欲望再回去用它"——不是质量问题，是工作流偏好问题
- 他更喜欢在终端/Claude Code 环境里完成一切

**选择**
- 不是"批评 Claude Design 不好"
- 而是将 Claude Design 的系统提示词和组件机制拆解后，做成 Claude Code 内的 Skill
- 定位为 GUI 工具的替代：能出 HTML、MP4、GIF、可编辑 PPTX

**结果**
- 19,900+ GitHub Stars（其第二大 Star 项目）
- 对平台无关：可在 Claude Code、Cursor、Codex 中使用

**体现的价值观**
- "不迁就工具，让工具适应工作流"——他对工具选择的核心逻辑
- 快速逆向工程 + 再创造，而不是等待官方支持

## 内容变现策略分析（综合）

| 层级 | 形式 | 定价 | 逻辑 |
|------|------|------|------|
| 免费基础设施 | 橙皮书（12本）、GitHub Skills | 免费 | 最大化传播，建立品牌 |
| 轻度付费社群 | 知识星球"AI编程：从入门到精通" | 365元/年，1900+成员 | 过滤严肃学习者，提供问答支持 |
| 企业定制 | 企业AI培训/咨询 | 2-10万/项目 | 高价值少量转化 |
| 演讲 | 大学/企业演讲 | 5-8万/场 | 被动收入，品牌驱动 |
| App 收费 | 小猫补光灯 Pro | 1元/下载 | 象征性收费，以上榜为目标 |

**核心变现逻辑**：免费内容 → 品牌背书 → 高价服务漏斗。并非传统知识付费的"用免费引流，付费出干货"，而是"免费就是干货，用影响力换高单价机会"。

## 决策中体现的独特价值观（总结）

1. **脆弱性即自由**：主动维持经济不安全状态，以此换取行动自由
2. **工具服务工作流**：不迁就任何工具，包括 Anthropic 自己的产品
3. **传播优先于变现**：在信息产品上，选择影响力，让高价值机会自动涌来
4. **基础设施思维**：做可被无限复用的系统（Skills），而不是一次性内容
5. **反直觉验证**：愿意推翻自己的积累（93KB→22KB），以数据说话
6. **身份认同清晰**："AI Native Coder" 是定义，不是辩护——对质疑选择正面对抗
7. **祛魅+重新定价**：对"出书"、"管理岗"、"大厂"等社会期待逐一祛魅，然后选择自己的路
